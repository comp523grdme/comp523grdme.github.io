Use Cases

Standard User

    for this application there is really only one type of user, a standard user; there is also no adminstrative function as all operations are initiated by the client

  • Install extension

  • On first launch, generate keys

    • import/get peer keys

      • keyserver

      • manually

  • Sharing your key

    • button in extension drop-down to copy longterm identity key to clipboard
    • Paste this key somewhere public on the internet

  • Send a message

    • All messages are sent to groups, even if it message is for one person

    • type text in popup

    • select contact groups below text box

    • ctrl + e or click button to copy to clipboard [close popup]

    • paste from clipboard

    • send to group

  • Receive a message

    • All messages are received by the individuals who are part of the group the message was sent to

    • See a message surrounded with ~~GrdMe~~ ~~!GrdMe~~

    • Attempt to decrypt with set of keys to see if message meant for user

  • Export a message to the database

    • go to menu

    • click export

    • get file browser

    • save in location of your choice

      • if encrypted save encrypted, if not save not

  • Stretch Features

    • add options link in key manager menu

    • change shorcut command

    • multikey support
